# Voyara - Modern Travel Booking Website

A modern, responsive travel booking site built with HTML, CSS, and JavaScript.

## What's included
- Destination search + combined filters (trip type, budget).
- Image-based destination cards.
- User sign up and login pages.
- Client-side authentication with session-aware navigation (localStorage).
- Booking call-to-action that requires login.

## Run locally
```bash
python3 -m http.server 8000
```
Then open:
- `http://localhost:8000/index.html`
- `http://localhost:8000/signup.html`
- `http://localhost:8000/login.html`

## Demo account flow
1. Sign up with name, email, and password.
2. You are redirected to the main page as a logged-in user.
3. Use search/filters and click **Book Now**.
4. Logout from the top-right navigation.
